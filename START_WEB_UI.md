# 🌐 WebCrawler Pro Web Dashboard

## 🚀 Quick Start

### Step 1: Open Terminal and Navigate
```bash
cd /Users/ceydaakin/BLG480E_Project2
```

### Step 2: Start Web Dashboard
```bash
python3 run_web_dashboard.py
```

### Step 3: Open Browser
Visit: **http://127.0.0.1:8080**

## 🎯 What You'll See

### 📊 Dashboard Features:
- **System Status**: Real-time crawler statistics
- **Performance Metrics**: Success rates, queue status
- **Live Search**: Search your indexed content in a web form
- **Auto-refresh**: Updates every 30 seconds
- **Beautiful UI**: Modern web interface with responsive design

### 🔍 Web Search Interface:
- Enter search queries in the web form
- See results in a table with clickable links
- Real-time search through your crawled content
- JSON API endpoints available

## 📱 Interface Types Available:

1. **Web Dashboard** (Browser): http://127.0.0.1:8080
2. **CLI Interface** (Terminal): `python3 -m src.main --help`

## ⚡ Quick Demo:

1. Start web dashboard: `python3 run_web_dashboard.py`
2. Open http://127.0.0.1:8080 in browser
3. Try searching for "example" or "HTTP"
4. View real-time system metrics

## 🛑 To Stop:
Press `Ctrl+C` in the terminal running the web server.
