QUICK START INSTRUCTIONS FOR TEACHERS/REVIEWERS:
===============================================

1. Clone the repository:
   git clone https://github.com/ceydaakin/web-crawler-multiagent.git
   cd web-crawler-multiagent

2. Install dependencies:
   pip install -r requirements.txt aiohttp-cors

3. Initialize database:
   python scripts/init_db.py

4. Start Web Dashboard:
   export PYTHONPATH=$(pwd)
   python run_web_dashboard.py

5. Open browser: http://127.0.0.1:8888

6. Test crawling (new terminal):
   export PYTHONPATH=$(pwd)
   python -m src.main index --origin "https://example.com" --depth 2

7. Test search:
   python -m src.main search --query "example"
